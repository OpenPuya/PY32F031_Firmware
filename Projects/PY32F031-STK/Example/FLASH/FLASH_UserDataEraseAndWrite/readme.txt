功能描述：
此样例演示了UserData擦除和写功能。

Function descriptions:
This example demonstrates the UserData erase and write functions.
================================================================================
测试环境：
测试用板：PY32F031_STK
MDK版本： 5.28
IAR版本： 9.20
GCC 版本：GNU Arm Embedded Toolchain 10.3-2021.10

Test environment:
Test board: PY32F031_STK
MDK Version: 5.28
IAR Version: 9.20
GCC Version: GNU Arm Embedded Toolchain 10.3-2021.10
================================================================================
使用步骤：
1. 编译下载程序到MCU，并运行；
2. 按下按键，程序进行UserData擦除和写功能；
3. UserData擦除成功和写成功，LED亮。

Example execution steps:
1. Compile and download the program to MCU and run it;
2. Press the button to perform UserData erase and  write functions in the program;
3. The UserData is successfully erased and is successfully written, and the LED
is on.
